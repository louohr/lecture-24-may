Block element vs Inline Element

Block element always takes the entire width of the page. That means there is no room for other elements on the same line, therefore the next element will be underneath. Examples are sections, articles, div, nav, header, and many more.

Inline elements are the opposite, they only take up width that correspons to the width of the content of the element. Examples are a-elements and span-elements.
